# P3DX Dashboard

A frontend prototype for governed cross-border data exchange, built with React and Tailwind CSS.

## Features

- World and gateway navigation
- Searchable agent and locker directory
- Connection states: Requested, Pending, and Established
- Minimal-data connection requests
- Consent grant, rejection, and revocation
- Searchable transaction history
- Responsive dashboard layout

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:5173`.

Create a production build with:

```bash
pnpm build
```

## Note

This is a frontend demonstration using sample data stored in memory. Refreshing the page resets changes, and no real personal data is exchanged.
